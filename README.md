N64-GBA
=======

An embarrassingly slow GBA emulator for the N64 that only runs ARMWrestler

Why?
----
Why not?


Download
--------
Check Releases on GitHub.

Features
--------
- Mode 4
- KEYINPUT
- DISPSTAT

Missing
-------
- All other graphics modes
- Interrupts
- Timers
- DMA
- Sprites
- Usefulness

Building
--------
- Install libdragon (tested with the version from the AUR)
- Build with CMake
